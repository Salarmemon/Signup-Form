# Sign Up Form

A responsive and modern sign-up form project built as part of the Intermediate HTML and CSS course from The Odin Project. This project focuses on form structure, styling, layout design, and basic form validation using pure HTML and CSS.

## 🚀 Features

- Responsive layout
- Custom styled form inputs
- Password confirmation fields
- Form validation using HTML attributes
- Clean and modern UI design
- Mobile-friendly structure
- Flexbox-based layout

## 🛠️ Built With

- HTML5
- CSS3
- Flexbox

## 📚 What I Learned

Through this project, I practiced:

- Creating accessible HTML forms
- Using semantic HTML elements
- Styling forms with CSS
- Working with Flexbox layouts
- Using pseudo-classes like `:focus` and `:invalid`
- Improving responsive design skills
- Managing spacing, typography, and visual hierarchy
